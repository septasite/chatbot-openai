# chatbot-openai

example openai in cli

## Required:

- nodejs version 17+

## Cara Install:

- masukkan API key di line 6, API key bisa anda dapatkan [disni](https://beta.openai.com/account/api-keys)
- buka cmd / powershell / git bash / etc
  > npm install / yarn
- untuk menjalankan
  > npm start / yarn start
